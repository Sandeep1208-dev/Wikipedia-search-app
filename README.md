# Wikipedia Search App

A simple web app that lets you search Wikipedia articles in real time using the Wikipedia Search API.

---

## 🔗 Live Demo
[View Live](https://github.com/Sandeep1208-dev/Wikipedia-search-app.git)

---

## 📸 Features

- Search Wikipedia articles by typing and pressing Enter
- Displays results with title, URL, and description
- Clickable links that open articles in a new tab
- Loading spinner while fetching results
- Results clear automatically on each new search

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Wikipedia Search API (https://apis.ccbp.in/wiki-search)

## 🔍 How It Works

1. User types a search query in the input box
2. On pressing Enter, a GET request is sent to the Wikipedia Search API
3. A loading spinner is shown while the request is in progress
4. Results are displayed with title, link, and description
5. Each result opens the Wikipedia article in a new tab
